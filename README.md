# Discord Bot Builder

Scratchのようにブロックを組み合わせるだけでDiscord Botのコードを自動生成するWebツール。

🌐 **ライブデモ**: https://yourusername.github.io/discord-bot-builder

## ✨ 特徴

- 🧱 **ビジュアルプログラミング** - ドラッグ＆ドロップでブロックを配置
- 💻 **多言語対応** - JavaScript / TypeScript / Python
- 📁 **テンプレート機能** - ウェルカムBot、音楽Botなど5種類
- 👁️ **リアルタイムプレビュー** - 生成コードを即時確認
- 🚀 **ワンクリックエクスポート** - すぐに実行可能

## 🚀 使い方

1. **左パネル**でブロックを組み立てる
2. **中央パネル**でコードを確認
3. **右パネル**でエクスポート
4. **自分でホスティング**して実行

## 🛠️ 対応機能

- メッセージ応答とコマンド処理
- リアクション追加と管理
- 埋め込みメッセージ作成
- メンバー参加・退出通知
- 変数操作・条件分岐・ループ

## 📁 プロジェクト構成

シンプルな静的サイト構成。サーバー不要で動作します。
